# Zero Pointer

## Описание
Это простая игра-платформер на Python и pygame.

## Как играть
Всего 15 уровней
Собирайте флажки и прыгайте
Управление:
Ходьба влево: A / <
Ходьба вправо: D / >
Прыжок: Space / ^
Взять флажок: Enter
На развернуть на весь экран / свернуть в окно: F11
Прыгайте по коричневым платформам
Серая платформа падение
Синий портал начало уровня
Серый портал нерабочий зеленый портал
Зеленый портал конец уровня
